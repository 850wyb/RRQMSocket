#### 介绍
RRQMSocket是一个整合性网络通信框架，特点是支持高并发、事件驱动、易用性强、二次开发难度低等。其中主要内容包括:TCP服务通信框架、文件传输、RPC等内容，后续还会继续扩展其他组件，希望大家多多支持。

#### 安装教程

1.  VS2017及以上，能正常联网，访问Nuget管理器
2.  选中项目->右击引用->管理Nuget程序包->搜索RRQMSocket->安装即可
3.  或者下载本源码，直接编译使用即可。
